# AGENTS.md

This file provides guidance to Codex when working with code in this repository.

## Specialist Guidance

Use the repo-local `sdk-engineer` plugin for implementation work in this SDK:

- feature work in `src/client.ts`
- OpenAPI regeneration and type-layer changes
- locale-filtering behavior in `src/localize.ts`
- test additions and build verification

## Commands

```bash
mise run generate      # Regenerate src/generated/api.ts from openapi.yaml
mise run build         # Compile to dist/ (ESM + CJS)
mise run test          # Run tests once
mise run test:watch    # Run Vitest in watch mode
mise run dev           # Build in watch mode
mise run ci            # generate -> build -> test

npm run generate       # Same as mise run generate
npm run build          # Same as mise run build
npm test               # Same as mise run test
npm run prepare        # generate -> build; run before committing source changes
```

## Architecture

The SDK has two layers:

```text
openapi.yaml
    |  npm run generate
    v
src/generated/api.ts     machine-generated types; never hand-edit
    |
    v
src/client.ts            public wrapper methods
src/localize.ts          deep locale-filter utility
src/types.ts             public config and params
src/index.ts             re-exports only
```

`src/generated/api.ts` is generated by `openapi-typescript` and consumed by `openapi-fetch` for typed HTTP calls. The public API surface lives in `src/client.ts`.

Each wrapper method should:

1. Source `siteId` and auth configuration from the client config.
2. Call the typed `openapi-fetch` client.
3. Unwrap `{ data, error, response }` and throw with the API error message on failure.
4. Apply `localize()` only when a locale is resolved.

## Critical Conventions

- Use explicit `.js` extensions in every relative import under `src/`. This project uses `module: "NodeNext"`.
- Do not hand-edit `src/generated/api.ts`; regenerate it from `openapi.yaml`.
- `src/generated/` is gitignored, so run `npm run generate` before `npm run build` in a fresh checkout.
- `dist/` is committed because this package is installed directly from GitHub. Run `npm run prepare` and commit `dist/` with source changes.

## Testing

- `src/localize.test.ts` covers the recursive locale transform directly.
- `src/client.test.ts` uses MSW to verify request construction, error handling, and locale filtering.
- New endpoint wrappers should get request-shape coverage, locale behavior coverage, and at least one failure-path test.
